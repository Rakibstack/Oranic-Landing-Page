# 🌿 Organic Landing Page

A modern, fully responsive organic skincare landing page built using **Next.js 14 (App Router)** and **Tailwind CSS**.  
This project focuses on clean UI, performance optimization, and responsive design.

---

## 🚀 Live Demo

🔗 Live URL: https://oranic-landing.vercel.app/ 
🔗 GitHub Repository: https://github.com/Rakibstack/Oranic-Landing-Page.git

---

## 📌 Features

- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
- ✅ Optimized Images using Next.js `<Image />`
- ✅ Google Font Optimization using `next/font`
- ✅ Clean Component-Based Architecture
- ✅ Floating Product Card UI
- ✅ Gradient Overlay for Better Text Visibility
- ✅ Reusable Layout (Navbar + Footer)

---

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Font Optimization:** next/font (EB Garamond)
- **Deployment:** Vercel
